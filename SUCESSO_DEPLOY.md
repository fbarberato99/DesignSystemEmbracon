## 🔧 Se o Deploy Falhar

### Verificar Node.js no Workflow
O workflow já está configurado para usar Node.js 22 (compatível com Storybook 10).

### Verificar Permissões
Certifique-se de que as permissões do workflow estão em "Read and write permissions".

### Re-executar Manualmente
Se necessário, você pode re-executar o workflow:
1. Acesse: https://github.com/fbarberato99/DesignSystemEmbracon/actions
2. Clique no workflow que falhou
3. Clique em "Re-run all jobs"
